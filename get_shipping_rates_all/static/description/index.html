This module allows you to fetch all shipping method cost at the time of adding shipping cost to the sale order.

This module is very useful to compare shipping cost of all shipping carrier you are using. this module fetch all the shipping cost and listed in table so you can compare the shipping cost of various shipping provider. this way you can provide cheaper rate for the shipping to your customers. Also you don't need to select shipping method one by one and compare the cost it give all shipping method cost in one click so it will save your time.

Features

  • Allow to compare all shipping method cost in one click.
  • You can select cheaper rate for your customers.
  • Time saving using this module you will get all shipping method cost in one click so you don't need to check one by one cost of shipping method.
  • Multi-company supported.
  • You can see the odoo default ADD SHIPPING button.


  • On click on ADD SHIPPING button it will open a popup. You will notice delivery method selection removed from view and new button added GET RATES.


  • On click on GATE RATES button it will check all shipping method available in system and fetch rate of every shipping method. if there is any warnings or errors while fetching rates you will see on screen.


  • Once all rates fetched you can select shipping method according to your need. it will allow to select one shipping method as per odoo standard. if there is any error while fetching shipping rates it will not allow to select that shipping method. Once you select shpping method you can simply click add button to apply shipping cost on sale order.


  • Shipping cost added to the sale order.
